# AI 求职助手

> 上传简历和岗位描述，AI 自动帮你生成优化简历、面试 PPT、面试讲稿和面试提问。

## 它能做什么？

| 功能 | 说明 |
|------|------|
| 优化简历 | 根据岗位要求，自动改写你的简历，突出匹配的技能和经历 |
| 面试 PPT | 自动生成一份精美的面试展示 PPT，帮你梳理自我介绍和项目亮点 |
| 面试讲稿 | 逐页生成 PPT 对应的演讲台词，照着念就行 |
| 面试提问 | 预测面试官可能问的问题，并给出回答思路 |

## 怎么用？

### 第一步：安装

打开电脑的"终端"（Mac）或"命令提示符"（Windows），复制粘贴下面这行命令，按回车：

```bash
pip install -r requirements.txt
```

### 第二步：设置 API Key

你需要一个 OpenAI 的 API Key（在 [OpenAI 官网](https://platform.openai.com/api-keys) 注册获取）。

**方法一（推荐）：在网页里输入**

启动后，在网页左侧展开"API 设置"，直接输入你的 Key。

**方法二：设置环境变量**

```bash
# Mac / Linux
export OPENAI_API_KEY="sk-你的密钥"

# Windows
set OPENAI_API_KEY=sk-你的密钥
```

### 第三步：启动

```bash
python run.py
```

### 第四步：打开浏览器

在浏览器地址栏输入：

```
http://localhost:8000
```

然后上传你的简历（支持 .docx 或 .txt），粘贴或上传岗位 JD，点击"开始生成"即可。

## 常见问题

**Q: 需要什么版本的 Python？**
A: Python 3.9 或以上版本。

**Q: 支持哪些 AI 模型？**
A: 默认使用 OpenAI 的 gpt-4o，也可以在网页的 API 设置里换成其他兼容的模型（如 DeepSeek、通义千问等）。

**Q: 会泄露我的个人信息吗？**
A: 不会。你的简历和 API Key 只在本地处理，不会上传到任何第三方服务器（除了你选择的 AI 服务商）。
